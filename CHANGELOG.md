# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-13

### Added
- Initial release
- Branching dialogue tree system with conditional logic
- Flag-based state management (game flags and conversation flags)
- Action system for flag manipulation and custom callbacks
- Dynamic text interpolation
- Event hooks for dialogue lifecycle events
- State serialization and deserialization
- History tracking with back navigation
- I18n adapter interface for internationalization support
- Comprehensive dialogue validation
- Full TypeScript support with complete type definitions
- Comprehensive test suite with 214 tests
- Interactive demo page with automated tests
